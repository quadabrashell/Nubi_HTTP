# DirectAccess
HTTP server.


### Setup
```
npm install
```

### Usage
Start server:
```
npm run start port=8080 path="./relative/path"
```

### Development
Start with nodemon:
```
npm run dev port=8989 path="./dev/path"
```

### Build project & start
```
npm run build
npm run serve port=80 path=www
```
